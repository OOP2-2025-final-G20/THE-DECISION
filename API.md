 #### ベースURL:
http://[サーバーPCのIP]:8000 通信形式: JSON

#### 1. お題と選択肢の取得 (GET)
アプリ起動時に、今表示すべき「お題」と「ボタンのラベル名」を取得します。

エンドポイント: /api/get_question

レスポンス例:
```bash
JSON

{
  "question": "一生食べるならどっち？",
  "optionA": "高級寿司",
  "optionB": "至高の焼肉"
}
```
#### 2. 投票の送信 (POST)
ユーザーがボタンをクリックしたときに、選択したデータをサーバーへ送ります。

エンドポイント: /api/vote

リクエストボディ:
```
JSON

{
  "choice": "A" 
}
(※ "A" または "B" を文字列で送信)

成功時のレスポンス例:

JSON

{
  "success": true,
  "current_votes": {
    "A": 15,
    "B": 12
  }
}
```
#### 3. 集計結果の取得 (GET)
最新の得票数だけを確認したい場合に使用します（リアルタイム更新用）。

エンドポイント: /api/results

レスポンス例:
```
JSON

{
  "A": 15,
  "B": 12,
  "total": 27
}
```
