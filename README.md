# THE-DECISION

## アプリの概要
特定の選択肢に対して、AorBのように投票を行う。投票を行ったら、待機画面に行き全員が終わったら自分で開示ボタンを押すと全員の解答が見えるようになる。また、過去の履歴の表示や選択肢の追加などをできるようにする。
## Usage：使い方
> このリポジトリのアプリを動作させるために行う手順
```bash
### 仮想環境（env）を作成
python3 -m venv env

### 仮想環境を有効化 (Mac / Linux)
source env/bin/activate
pip install -r requirements.txt
python base.py
```
